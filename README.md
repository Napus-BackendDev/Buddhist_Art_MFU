# 🪷 Buddhist-Art

A mini full-stack project using **Vue.js**, **NestJS**, and **Docker** for local development.

---

## 📁 Project Structure

```
📦 Buddhist-Art
├── backend/
│   ├── app/        # NestJS API
├── frontend/
│   ├── app/        # Vue User Panel
```

---

## ⚙️ Installation & Setup

### 1. Install root dependencies

```bash
npm i
```

### 2. Install dependencies

```bash
npm run setup
```

### 3. Start a service

```bash
npm run start
```

> Choose **one** service to run in the same terminal:
- `🖼️ Frontend (Vue)`
- `🔧 Backend (NestJS)`
